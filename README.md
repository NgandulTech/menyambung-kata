# ⚡ Sambung Kata Pro

**Sambung Kata Pro** adalah website pencari kata KBBI yang dirancang khusus untuk membantu kamu memenangkan game **Sambung Kata v.Indo** di Roblox!

## 🌐 Try It Live

👉 **[[(https://ngandultech.github.io/menyambung-kata/)](https://ngandultech.github.io/menyambung-kata/)]**

## ✨ Fitur

- 🔍 **Pencarian Cepat** — Cari kata berdasarkan awalan, akhiran, atau keduanya sekaligus. Hasil langsung muncul tanpa perlu tekan Enter.
- 🎯 **Dual Search** — Ketik awalan + akhiran bersamaan, otomatis tampil 2 section side-by-side: hasil kombinasi dan semua kata berawalan.
- 🔤 **Alphabet Bar** — Navigasi cepat berdasarkan huruf abjad dengan jumlah kata per huruf.
- 🧠 **Mode Strategi** — Masukkan kata terakhir lawan untuk menemukan kata terbaik yang membuat lawan kesulitan.
- ➕ **Tambah Kata Custom** — Tambahkan kata yang belum ada di KBBI langsung dari halaman pencarian.
- 👁️ **Sembunyikan Kata** — Sembunyikan kata yang tidak valid di game ke section tersembunyi.
- 💾 **Data Persisten** — Kata custom dan kata tersembunyi disimpan di browser (localStorage).

## 📊 Dataset

- **114.000+ kata** dari Kamus Besar Bahasa Indonesia (KBBI) Edisi VI
- Sudah dioptimasi untuk pencarian cepat dengan prefix & suffix indexing

## 🛠️ Tech Stack

- HTML, CSS, JavaScript (Vanilla — tanpa framework)
- LocalStorage untuk persistensi data
- GitHub Pages untuk hosting

## 🚀 Menjalankan Secara Lokal

```bash
npx -y http-server -p 8080
```

Buka `http://localhost:8080` di browser.
